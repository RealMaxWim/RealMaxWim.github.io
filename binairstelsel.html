Het binaire stelsel is een getallenstelsel wat in tegenstelling tot ons decimale getallen stelsel (getallen 0-9) gebruikt maakt van de getallen 0 en 1. Het binairestelsel wordt gebruikt in computers om makkelijker schakelingen uit te voeren. Dit kan met de getallen 1 of 0, die voor een schakeling kunnen zorgen. We gebruiken het binairstelsel bij informatica, omdat het dus goed geschikt is voor computers om informatie te verwerken en schakelingen uit te voeren.

Een bit is een eenheid die twee waarden kan aanemen (aan-uit, ver-dichtbij). In het Binairestelsel gebruikt men bits met waarde 0 of 1. Een byte is ook een binaire eenheid. Het ASCII (American Standard Code for Information Exchange) is de universele code die gebruikt wordt voor dagelijkse tekens, zoals getallen letters en speciale tekens. Het ASCII stelsel werkt op 7 bits. Zie de afbeedling hiernaast voor wat voorbeelden voor tekens die in het ASCII stelsel zitten.
